What is a List in java?
A List is a collection that:
	-> Stores the multiple values
	-> Maintains the insertion order
	-> Allows Duplicate values
List is an interface that is present in java.util package.

Important List Methods:
	1) Add Method: list.add(10);
	2) Add (index,element): list.add(2,10);
	3) Access Method(get(index)): list.get();
	4) Replace element (set(idx,value)): list.set(1,25);
	5) Remove element (remove(idx), remove(Integer.valueOf(value))): list.remove(0);// idx
									 list.remove(Integer.valueOf(20)); //by value
	6) Size (size(list)): list.size();
	7) Contains: list.contains();
	8) isEmpty: list.isEmpty();
